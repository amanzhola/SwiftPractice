# 🧩 Project 5 – WordScramble

A SwiftUI word game where users create new words from a given root word, with validation and scoring logic.

---

## 🎮 Features

- Generate a random root word
- Enter new words based on the root word
- Validate words with multiple rules:
  - must be original
  - must be possible from given letters
  - must be a real English word
- Display list of used words
- Show word score and feedback
- Load words from local file (`start.txt`)

---

## 🧠 What I Learned

- Working with user input in SwiftUI
- Validating strings with custom logic
- Using `UITextChecker` for spell checking
- Loading data from local files (`Bundle`)
- Managing dynamic lists
- Handling errors and showing alerts
- Building a simple game with logic + UI

---

## 🛠 Technologies

- SwiftUI
- String manipulation
- UITextChecker
- Bundle file loading
- Lists & Forms
- State management

## 🧩 Проект 5 – WordScramble

Игра на SwiftUI, в которой пользователь составляет слова из заданного слова с проверкой и логикой подсчёта.

---

## 🎮 Возможности

- Генерация случайного слова
- Ввод новых слов
- Проверка слов:
  - слово должно быть новым
  - слово должно составляться из букв исходного слова
  - слово должно существовать в языке
- Отображение списка введённых слов
- Подсчёт очков
- Загрузка слов из файла (`start.txt`)

- Показывать ошибки через alert
