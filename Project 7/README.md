# 💸 Project 7 – iExpense

A SwiftUI app for tracking personal expenses with data persistence and categorized spending.

---

## 🚀 Features

- Add new expenses
- Delete expenses
- Categorize expenses (Personal / Business)
- Display expenses in a list
- Persist data using UserDefaults
- Highlight large expenses visually
- Improved UI in Pro version

---

## 🧠 What I Learned

- Managing app state with `@State` and `@ObservableObject`
- Storing data using `UserDefaults`
- Encoding and decoding with `Codable`
- Creating dynamic lists with `ForEach`
- Deleting items with swipe actions
- Passing data between views
- Structuring multi-screen SwiftUI apps

---

## 🛠 Technologies

- SwiftUI
- UserDefaults
- Codable
- NavigationStack
- Lists & Forms
