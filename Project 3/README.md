# 🎯 Project 3 – SwiftUI Fundamentals

A collection of beginner SwiftUI apps demonstrating core concepts such as UI layout, user interaction, and basic game logic.

---

## 🚀 Included Apps

### 🇩🇪 GuessTheFlag

A quiz game where users guess the correct country flag.

#### Features:
- Random flag selection
- Score tracking
- Alerts for correct/incorrect answers
- Simple and engaging UI

---

### ✂️ RockPaperScissor

A game where users must either win or lose against the app based on given instructions.

#### Features:
- Random opponent choice
- Dynamic game rules (win or lose)
- Score tracking
- Interactive UI

---

### 💰 WeSplit

A bill-splitting app that calculates how much each person should pay.

#### Features:
- Enter check amount
- Select number of people
- Add tip percentage
- Calculate total per person

---

## 🧠 What I Learned

- Building UI with SwiftUI
- Using `@State` for managing data
- Creating reusable views
- Handling user input
- Displaying alerts
- Building simple game logic
- Working with forms and pickers

---

## 🛠 Technologies
