# 🎬 Project 6 – Animations

A SwiftUI project focused on building animations and transitions, progressing from simple effects to more advanced interactive UI.

---

## 🚀 Features

- Basic animations using SwiftUI
- Interactive animations triggered by user actions
- Smooth transitions between views
- Scaling, rotation, and opacity effects
- Custom animation timing and curves
- Building engaging UI with motion

---

## 🧠 What I Learned

- Using `withAnimation`
- Understanding implicit vs explicit animations
- Working with animation modifiers
- Creating smooth transitions
- Handling state changes with animations
- Designing interactive UI with motion
- Improving user experience with visual feedback

---

## 🛠 Technologies

- SwiftUI
- Animations
- State management
- View modifiers
- Transitions
