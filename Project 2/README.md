# 🇩🇪 Project 2 – GuessTheFlag

A SwiftUI quiz game where users guess the correct country flag, built step-by-step from a simple UI to a complete interactive game.

---

## 🚀 Features

- Display random country flags
- Ask user to select the correct flag
- Show score updates
- Display alerts for correct/incorrect answers
- Randomize questions each round
- Improved UI and logic across versions

---

## 🧠 What I Learned

- Building UI with SwiftUI stacks
- Using `@State` to manage game state
- Handling user interaction (buttons)
- Displaying alerts
- Randomizing data
- Structuring a simple game
- Improving UI step-by-step

---

## 🛠 Technologies

- SwiftUI
- State management
- Buttons & Images
- Alerts
- Randomization
- Layout (VStack, HStack, ZStack)
