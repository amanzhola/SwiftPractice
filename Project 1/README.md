# 💰 Project 1 – WeSplit

A SwiftUI app that calculates how to split a bill between multiple people, including tips.

This is the first project in my SwiftUI learning journey, built step-by-step from a basic layout to a complete working app.

---

## 🚀 Features

- Enter total bill amount
- Select number of people
- Choose tip percentage
- Calculate total per person
- Dynamic updates based on input
- Improved UI and logic across versions

---

## 🧠 What I Learned

- Building UI with SwiftUI
- Using `@State` to manage input
- Working with `TextField`
- Using `Picker` and `Form`
- Performing calculations in real time
- Handling user input
- Structuring a simple app

---

## 🛠 Technologies

- SwiftUI
- State management
- Forms
- TextField
- Picker
- Basic calculations

## 💰 Проект 1 – WeSplit

Приложение на SwiftUI для разделения счета между людьми с учётом чаевых.

Это первый проект в моём обучении SwiftUI, созданный шаг за шагом — от простого интерфейса до полноценного приложения.

---

- Ввод суммы счета
- Выбор количества людей
- Выбор процента чаевых
- Расчет суммы на человека
- Автоматическое обновление результата
- Улучшение UI и логики по мере развития
