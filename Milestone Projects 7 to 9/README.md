# ✅ Milestone Projects 7 to 9 – HabitTracker

A SwiftUI habit tracking app where users can create habits, view details, and monitor their progress.

---

## 🚀 Features

- Add new habits
- View a list of saved habits
- Open a detailed habit screen
- Track personal progress
- Clean SwiftUI navigation flow
- Improved UI in the updated version with custom background and reusable view modifiers

---

## 🧠 What I Learned

- Building multi-screen apps with SwiftUI
- Passing data between views
- Structuring app screens clearly
- Creating reusable UI components
- Improving interface design with custom backgrounds and modifiers
- Organizing project versions from base to improved UI

---

## 🛠 Technologies

- SwiftUI
- State management
- NavigationStack / Navigation
- Custom Views
- ViewModifiers

# 🇷🇺 Русская версия

## ✅ Milestone Projects 7 to 9 – HabitTracker

Приложение на SwiftUI для отслеживания привычек, где пользователь может создавать привычки, открывать детали и следить за своим прогрессом.

---

## 🚀 Возможности

- Добавление новых привычек
- Просмотр списка привычек
- Открытие детального экрана привычки
- Отслеживание личного прогресса
- Чистая навигация между экранами
- Улучшенный интерфейс в версии `updated` с кастомным фоном и переиспользуемыми модификаторами

---

## 🧠 Чему я научился

- Создавать многоэкранные приложения на SwiftUI
- Передавать данные между экранами
- Правильно разделять экраны по ответственности
- Создавать переиспользуемые UI-компоненты
- Улучшать интерфейс через фон и собственные modifiers
- Вести базовую и улучшенную версии проекта

---

## 🛠 Технологии

- SwiftUI
- Управление состоянием
- NavigationStack / Navigation
- Custom Views
- ViewModifiers
