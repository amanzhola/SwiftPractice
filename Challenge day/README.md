# 🔄 Challenge Day – Unit Converters

A collection of small SwiftUI apps for converting values between different measurement units.

This folder contains two converter apps built as practice projects:
- Temperature Converter
- Distance Converter

---

## 🚀 Included Apps

### 🌡️ Converter (Temperature)

A SwiftUI app for converting temperature values between different units.

#### Features
- Enter a numeric temperature value
- Select input unit
- Select output unit
- Instantly see converted result
- Simple and clean SwiftUI form interface

---

### 📏 Converter (Distance)

A SwiftUI app for converting distance values between measurement units.

#### Features
- Enter a numeric distance value
- Choose input unit
- Choose output unit
- Instantly calculate converted value
- Clean and reusable SwiftUI layout

---

## 🧠 What I Learned

- Building small utility apps with SwiftUI
- Working with `@State` for user input
- Using `TextField`, `Picker`, and `Form`
- Converting values between units
- Structuring simple apps clearly
- Practicing logic and UI together

---

## 🛠 Technologies

- SwiftUI
- State management
- Forms
- TextField
- Picker
- Unit conversion logic

# 🇷🇺 Русская версия

## 🔄 Challenge Day – Конвертеры единиц

Набор небольших приложений на SwiftUI для перевода значений между разными единицами измерения.

В этой папке находятся два мини-проекта:
- Конвертер температуры
- Конвертер расстояния

---

## 🚀 Приложения

### 🌡️ Converter (Temperature)

Приложение на SwiftUI для перевода температуры между разными единицами.

#### Возможности
- Ввод числового значения температуры
- Выбор исходной единицы
- Выбор итоговой единицы
- Мгновенный показ результата
- Простой и аккуратный интерфейс на `Form`

---

### 📏 Converter (Distance)

Приложение на SwiftUI для перевода расстояния между единицами измерения.

#### Возможности
- Ввод числового значения расстояния
- Выбор исходной единицы
- Выбор итоговой единицы
- Мгновенный расчёт результата
- Чистая и понятная структура интерфейса

---

- Работать с `TextField`, `Picker` и `Form`
- Делать перевод значений между единицами
- Строить простые приложения с логикой и UI
- Практиковать вычисления в реальном времени
