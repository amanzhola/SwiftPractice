# 🚀 Project 8 – Moonshot

A SwiftUI app that explores NASA Apollo missions, astronauts, and mission details using JSON data and custom UI layouts.

---

## 🌌 Features

- Browse Apollo missions
- View mission details
- Explore astronaut information
- Decode JSON data from local files
- Switch between list and grid layouts
- Custom dark theme UI
- Smooth navigation between screens

---

## 🧠 What I Learned

- Working with JSON and `Codable`
- Loading local data using `Bundle`
- Building reusable views
- Navigation between multiple screens
- Creating grid and list layouts
- Structuring larger SwiftUI apps
- Using custom colors and themes

---

## 🛠 Technologies

- SwiftUI
- Codable
- JSON decoding
- LazyVGrid
- NavigationStack
- Custom Views & Modifiers

## 📂 Project Structure

- `Moonshot(part_one)/` — basic layout and images
- `Moonshot(part_two)/` — JSON decoding and data models
- `Moonshot(part_three)/` — mission & astronaut views
- `Moonshot(part_four)/` — final version with grid/list UI and improvements
- `Images/` — early screenshots and assets

---

## 🎯 Summary

This project demonstrates:

- working with structured JSON data
- building scalable SwiftUI layouts
- separating data models and UI
- improving UI step-by-step across multiple versions
- creating a complete multi-screen app
