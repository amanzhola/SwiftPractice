# ✂️ Milestone Project – RockPaperScissors

A SwiftUI game where the player must either win or lose against the app based on given instructions.

---

## 🎮 Gameplay

The app randomly selects:
- a move (rock, paper, or scissors)
- a condition: **you must WIN or LOSE**

The player must choose the correct move to match the requirement.

---

## 🚀 Features

- Random game logic
- Dynamic instructions (win or lose)
- Score tracking
- Multiple rounds
- Game restart functionality
- Interactive UI with buttons

---

## 🧠 What I Learned

- Building game logic in SwiftUI
- Using `@State` for tracking score and rounds
- Working with randomness
- Handling user input
- Creating reusable UI components
- Structuring game flow (round → score → restart)

---

## 🛠 Technologies

- SwiftUI
- State management
- Randomization
- Buttons & UI interaction
- Game logic

# 🇷🇺 Русская версия

## ✂️ Milestone Project – RockPaperScissors

Игра на SwiftUI, в которой нужно выиграть или проиграть в зависимости от условия.

---

## 🎮 Игровая логика

Приложение случайно выбирает:
- ход (камень, ножницы, бумага)
- условие: **нужно выиграть или проиграть**

Игрок должен выбрать правильный вариант.

---

## 🚀 Возможности

- Случайная логика игры
- Динамические условия (выиграть / проиграть)
- Подсчёт очков
- Несколько раундов
- Возможность перезапуска игры
- Интерактивный интерфейс

- Обрабатывать действия пользователя
- Строить игровой цикл (раунд → результат → рестарт)
