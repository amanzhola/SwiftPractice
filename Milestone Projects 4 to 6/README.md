# ✖️ Milestone Projects 4 to 6 – MultiplyFun

A SwiftUI math quiz app designed to help users practice multiplication tables in a fun and interactive way.

---

## 🚀 Features

- Choose multiplication table range
- Select number of questions
- Start a custom quiz session
- Answer multiplication questions interactively
- Track score during the game
- Show final result at the end
- Simple and clean SwiftUI interface

---

## 🧠 What I Learned

- Building a small game in SwiftUI
- Working with `@State` to manage quiz progress
- Generating random questions dynamically
- Handling user input and answer validation
- Structuring game flow from setup to final score
- Designing a simple educational app UI

---

## 🛠 Technologies

- SwiftUI
- State management
- Forms
- Stepper
- Picker
- Alerts
- Random question generation

---

## 🚀 Возможности

- Выбор таблицы умножения
- Выбор количества вопросов
- Запуск своей игровой сессии
- Ответы на вопросы по умножению
- Подсчёт очков по ходу игры
- Показ финального результата
- Простой и аккуратный интерфейс

## 🧠 Чему я научился

- Создавать небольшую игру на SwiftUI
